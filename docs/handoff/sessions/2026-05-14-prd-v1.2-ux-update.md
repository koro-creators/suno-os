## Aprendizados / pegadinhas

- **Edit tool bloqueado em modo don't-ask**: todos os patches foram feitos via `python3 str.replace()` em scripts `/tmp/`. Padrão robusto — assertions antes de cada replace evitam double-replace.
- **Seção 13 (Progressive Disclosure) não tinha PX-06** mesmo após v1.1 — seção foi adicionada junto com PX-07/PX-08 nesta sessão.
- **Drive Suno ≠ Google Drive do cliente**: a confusão entre "Drive externo do cliente" e "Drive interno da Suno" é frequente nos docs — REST-08 v2 é o anchor normativo.
- **JN-15 e JN-17** existem no PRD Parte 3 (JN-15: Oráculo usado sem reunião, JN-17: Governança Setorial Sponsor) mas não foram incluídas no UX Inventário — essas jornadas não geram telas próprias ainda, são extensões de fluxos existentes.
