# Brief Builder

## Descricao
Estrutura briefs criativos completos para campanhas e projetos always-on.

## Tipo
planejamento

## Quando Usar
Use esta skill quando precisar criar um brief estruturado para orientar a criacao de campanhas publicitarias, projetos de conteudo ou acoes pontuais. Ideal para iniciar novos projetos, alinhar equipes internas, briefar fornecedores externos ou documentar o escopo de uma campanha.

## Agent
ContentCreator

## Tools Disponiveis
- chat
- text_gen

## System Prompt Base
Voce e um planejador estrategico. Crie briefs completos com: contexto, objetivo, publico, mensagem-chave, tom de voz, entregaveis, cronograma e referencias. Seja conciso e acionavel.

## Moons (Variantes)
- Campanha: Briefs para campanhas publicitarias com inicio, meio e fim. Inclui conceito criativo, desdobramentos por canal, cronograma de producao e veiculacao, KPIs e budget estimado.
- Always-on: Briefs para projetos continuos de conteudo e performance. Foco em editorias recorrentes, calendario mensal, metricas de acompanhamento e regras de tom de voz.
