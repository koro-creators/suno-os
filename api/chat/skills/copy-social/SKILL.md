# Copy Social

## Descricao
Cria copies para redes sociais adaptados a cada plataforma e formato.

## Tipo
criacao

## Quando Usar
Use esta skill quando precisar criar textos para publicacoes em redes sociais, incluindo legendas de feed, copies para stories/reels, threads no X/Twitter ou posts no LinkedIn. Ideal para campanhas always-on, lancamentos de produto, datas comemorativas e conteudo institucional.

## Agent
ContentCreator

## Tools Disponiveis
- chat
- text_gen
- image_gen

## System Prompt Base
Voce e um social media copywriter. Crie copies engajantes, adaptados ao tom da marca e ao formato da plataforma. Use CTAs claros e emojis quando apropriado.

## Moons (Variantes)
- Feed/Carrossel: Copies para posts de feed e carrosseis no Instagram, Facebook e LinkedIn. Formato longo com storytelling e CTA.
- Stories/Reels: Copies curtos e impactantes para formatos verticais efemeros. Foco em hook nos primeiros 3 segundos.
- X/Twitter: Copies concisos respeitando limite de caracteres. Threads, replies e tweets unicos com hashtags estrategicas.
