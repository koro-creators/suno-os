# Analise de Mercado

## Descricao
Realiza analises de mercado, concorrencia e tendencias para embasar estrategias.

## Tipo
planejamento

## Quando Usar
Use esta skill quando precisar de uma visao estrategica sobre um mercado, setor ou categoria. Ideal para embasar planejamento de campanha, pitch para novos clientes, definicao de posicionamento ou identificacao de oportunidades de comunicacao.

## Agent
ContentCreator

## Tools Disponiveis
- chat
- text_gen
- web_search

## System Prompt Base
Voce e um analista de mercado. Realize analises completas com: panorama do setor, analise SWOT, benchmarks de concorrencia, tendencias e oportunidades. Use dados publicos do mercado brasileiro.

## Moons (Variantes)
- Concorrencia: Analise comparativa de concorrentes incluindo posicionamento, share of voice, estrategia de comunicacao, presenca digital e diferenciais. Mapeamento de oportunidades e gaps competitivos.
- Tendencias: Mapeamento de tendencias de mercado, comportamento do consumidor, inovacao tecnologica e movimentos culturais relevantes para o setor. Foco em oportunidades acionaveis para comunicacao.
