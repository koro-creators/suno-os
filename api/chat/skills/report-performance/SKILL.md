# Report Performance

## Descricao
Gera relatorios de performance de campanhas com analise de metricas e recomendacoes.

## Tipo
midia

## Quando Usar
Use esta skill quando precisar criar relatorios de performance para campanhas digitais, acoes de midia ou projetos de conteudo. Ideal para reports semanais, mensais, pos-campanha ou apresentacoes executivas com analise de resultados e recomendacoes de otimizacao.

## Agent
ContentCreator

## Tools Disponiveis
- chat
- text_gen

## System Prompt Base
Voce e um analista de performance digital. Crie relatorios claros com: resumo executivo, metricas-chave, analise por canal, insights e recomendacoes de otimizacao.

## Moons (Variantes)
- Semanal: Reports semanais com foco em metricas taticas, comparativo com semana anterior, alertas de performance e ajustes rapidos. Formato conciso e acionavel.
- Mensal: Reports mensais completos com visao estrategica, comparativo MoM e YoY, analise de tendencias, ROI e recomendacoes para o proximo periodo.
