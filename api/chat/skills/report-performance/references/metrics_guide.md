## Metricas de Funil

### Topo de Funil (Awareness)
| Metrica | O que mede |
|---|---|
| Impressoes | Volume total de exibicoes |
| Alcance | Usuarios unicos impactados |
| CPM | Eficiencia de custo para awareness |
| Frequencia | Repeticao media por usuario |
| Brand Lift | Aumento de lembranca/reconhecimento |
| Share of Voice | Presenca vs concorrentes |

### Meio de Funil (Consideracao)
| Metrica | O que mede |
|---|---|
| Cliques | Volume de interesse ativo |
| CTR | Eficiencia do criativo em gerar interesse |
| CPC | Custo de cada interesse ativo |
| Tempo no site | Profundidade de engajamento |
| Pages/session | Navegacao no site |
| Video Views | Consumo de conteudo de video |
| Engajamento social | Curtidas, comentarios, compartilhamentos |

### Fundo de Funil (Conversao)
| Metrica | O que mede |
|---|---|
| Conversoes | Acoes de valor realizadas |
| CPA/CPL | Custo por acao/lead |
| CVR | Eficiencia de conversao |
| ROAS | Retorno financeiro sobre investimento |
| Revenue | Receita gerada |
| Ticket medio | Valor medio por conversao |
| LTV | Valor do cliente ao longo do tempo |

## Estrutura do Report

### Report Semanal
1. **Resumo executivo** (3-5 bullet points)
   - Budget gasto vs planejado
   - Principais KPIs vs meta
   - Alertas e destaques
2. **Metricas-chave** (tabela comparativa semana atual vs anterior)
3. **Top performers** (melhores criativos/audiencias)
4. **Alertas** (metricas abaixo do esperado)
5. **Acoes para proxima semana** (otimizacoes planejadas)

### Report Mensal
1. **Resumo executivo** (paragrafo + bullet points)
2. **Visao geral de investimento** (budget por canal, pacing)
3. **Resultados por canal** (metricas detalhadas com comparativo MoM)
4. **Analise de criativos** (performance por formato, mensagem, visual)
5. **Analise de audiencia** (segmentos, demographics, geography)
6. **Funil de conversao** (taxa de cada etapa)
7. **Insights e aprendizados** (o que funcionou e o que nao)
8. **Recomendacoes** (otimizacoes para proximo mes)
9. **Projecao** (estimativa de resultados mantendo otimizacoes)

## Dicas para Reports Acionaveis
1. **Comece pelo "so what"** — nao liste numeros, explique o que significam
2. **Compare sempre** — numeros isolados nao dizem nada (vs meta, vs periodo anterior, vs benchmark)
3. **Destaque anomalias** — o que subiu ou caiu significativamente e por que
4. **Termine com acoes** — cada insight deve gerar uma recomendacao
5. **Adapte ao publico** — report para C-level e diferente de report para operacao
6. **Visualize** — use graficos para tendencias, tabelas para comparacoes
7. **Seja honesto** — mostre o que nao funcionou e o plano de correcao
